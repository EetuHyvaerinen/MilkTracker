# MilkTracker

MilkTracker is an Android app to help parents track milk feedings and diaper changes for their baby.

## Demo video
<img src="demo/recording.gif" width="160" />


---

## Features

- Log milk feedings in ml
- Log diaper changes
- View history of feedings and diaper changes

---

## Requirements

Before running MilkTracker, make sure you have the following installed:

- Android Studio
- Java 11 or higher

---

## Installation

### 1. Clone the repository

    git clone https://github.com/EetuHyvaerinen/MilkTracker.git

### 2. Open in Android Studio

- Launch Android Studio
- open the cloned `MilkTracker` folder
- Let the project sync automatically

### 3. Run the app

- Start an emulator or connect a physical device
- Run the project in Android Studio
